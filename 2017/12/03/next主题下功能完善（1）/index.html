next主题下功能完善（1）

发表于 2017-12-03
字数统计: 443 | 阅读时长 ≈ 2

说明

这几天刚把博客搭建起来，看到next主题不错就选用了，接着研究next主题设置，中途遇到许多大大小小的问题，以此记录。

tips：请根据上一篇《hexo+github创建个人博客》第7点完成next主题替换。我的next版本5.12

1.语言设置

打开根目录下的_config.yml，找到language，设置为zh-Hans（language: zh-Hans），即可配置为中文 。
tips：冒号后面有空格，后面的配置都是这样。

2. 完善网站标题

打开根目录下的_config.yml，配置：

title: 小易的博客 #网站标题
subtitle: 越学越无法自拔 #个性签名
description: #博文描述
author: 小易 #博主名称

3.next样式替换

next有四种样式：

打开theme/next/_config.yml，找到如图
去掉#和把原来的加上#即可替换样式（选择自己喜欢的）。

4.设置头像

  1. 把你的头像图片放于themes\next\source\images下
  2. 打开theme/next/_config.yml
  3. 找到avatar
  4. 更改图片地址即可（第一次不显示，后面博主把图片格式改成和它一样就好了）

5.添加友情链接和社交连接

打开theme/next/_config.yml，找到如图：

1.social格式为：名称： 链接 || icon
更改即可，social_icons改为true
2.link格式： 名称： 地址

设置主页不显示全文

  1. 打开theme/next/_config.yml
  2. 找到auto_excerpt，改为true即可

添加RSS

  1. 打开git（git位置在你博客根目录）
  2. 执行npm install –save hexo-generator-feed，下载插件到node_modules
  3. 打开根目录下_config.yml

  4. 在下方添加:

    #Extensions  
##Plugins: http://hexo.io/plugins/  
    plugins: hexo-generate-feed

  5. 打开theme/next/_config.yml

  6. 找到rss，冒号后面添加atom.xml（即 rss: atom.xml），即可
