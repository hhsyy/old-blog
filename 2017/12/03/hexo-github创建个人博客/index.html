hexo+github创建个人博客

发表于 2017-12-03 |
字数统计: 657 | 阅读时长 ≈ 3

1.准备

- git（注册好github账号，下载对应系统版本git）
- node.js（百度下载安装）

2.安装hexo

1. 创建文件夹（自己定义位置与名称）
2. 单击右键Git Bash Here（需已安装好git）
3. 在命令窗口处输入： npm install -g hexo（创建的目录下，以下的命令都在其窗口操作）
4. 安装npm，命令： npm install
5. 安装完成可查看版本（hexo -v，node -v，npm -v）

3.创建github仓库

前提：你已注册

1. 打开你的github，新建新的仓库 
2. 在Repository name输入框需按照：yourname.github.io命名，后面默认创建即可（需public）
3. 如买域名（自行购买）请继续看，无则无需观看下面两点（github作为服务器）
4. 于你根目录下（你创建的目录）新建文件CNAME，无后缀，填入你的域名（如www.hhsyy.top）
5. 打开你创建的github仓库，点击该仓库的settings，下拉找到Custom domain填入你的域名。
6. 在你申请的域名网站上解析（http://yourname.github.io/）

4.搭建博客

前提：把git的user与email设置，ssh授权成功，教程自行百度

1. 初始化hexo：hexo init
2. 生成界面：hexo g（hexo generate）
3. 启动服务：hexo s（hexo server），此方法Ctrl+c后会404
4. 在浏览器输入：http://localhost:4000/（可以看到默认主题界面）
5. 更改根目录下（你创建的文件）的_config.yml ：
    deploy:
      type: git  
      repo: git@github.com:yourname/yourname.github.io.git
      branch: master
6.自动部署修改的文件：npm install hexo-deployer-git --save（无需Ctrl+c）
7.发布服务：hexo clean && hexo g && hexo d（hexo deploy）
8.如果配置了ssh不用授权，否则...

5.预览

在浏览器输入你的域名，打开，如没有则（http://yourname.github.io/）。

6.编写博客

1. 其为使用markdown，按照其规范编写。
2. 在根目录下输入hexo new "名称"创建md文件
3. 在根目录\source\_posts下找到你创建的文件打开（可下载markdown编译器打开编译）
4. 按照md语法编译，md教程直通车[http://www.markdown.cn/]

7.扩展（更换主题）

1. 选取主题，https://hexo.io/themes/
2. 点击名称进入github，找到clone or download，复制链接
3. 在命令窗口输入git clone 链接 （比如：git clone https://github.com/litten/hexo-theme-yilia.git themes/yilia），详情可参照git命令
4. 打开根目录下_config.yml，更改theme（如theme: yilia）
5. npm install hexo-deployer-git --save
6. hexo clean && hexo g && hexo d
7. 网页观看即可

tips：有些主题需下载插件，可以按照github你选择的主题下的教程操作。
